1. PROJECT SUMMARY
================================================================================

Vuyela Logistics is a comprehensive B2B logistics website built with Next.js 15,
designed to showcase the company's services, compliance credentials, and network
partnership opportunities. The site emphasizes authority, precision, and partnership
through a sophisticated design system using Deep Anthracite (#1a1a1a), Vuyela Gold
(#FACC15), and Clean White.

Key Brand Values:
- Reliability: 98.5% On-Time-In-Full delivery rate
- Integrity: Full compliance with NBCRFLI, GIT insurance, Transnet vendor standards
- Expertise: 20+ years of combined industry experience

   Headlines/Display: Playfair Display (Serif)
   - Weights: 400, 500, 600, 700
   - Usage: H1, H2, H3, brand statements
   - Association: Authority, Heritage, Elegance, Vision
   
   Body/UI: Inter (Sans-Serif)
   - Weights: 400-600
   - Usage: Body text, UI elements, links
   - Association: Clarity, Modernity, Readability, Trust
   
   Implementation: Next.js Font Optimization (next/font/google)
   - Eliminates Cumulative Layout Shift (CLS)
   - Fastest possible load times
   - CSS Variables: --font-playfair, --font-inter

   - Signals sophistication and confidence

   3. Value Propositions: Quantify Values
   4. Services/Industries: Detailed Solutions
   5. CTA: Conversion Points
   
   Responsive Breakpoints:

   - Desktop: lg: 1024px
   - Large Desktop: xl: 1280px
