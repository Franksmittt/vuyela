# Vuyela Logistics Website

A modern, high-performance Next.js website for Vuyela Logistics, built with the App Router and optimized for SEO, performance, and user experience.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Next.js Font Optimization (Inter)
- **Deployment**: Ready for Vercel

## Features

- ✅ Next.js App Router with React Server Components
- ✅ TypeScript for type safety
- ✅ Tailwind CSS for styling
- ✅ SEO optimized (robots.txt, sitemap.xml)
- ✅ Font optimization with next/font
- ✅ Modern project structure

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
├── app/              # App Router directory
│   ├── layout.tsx    # Root layout
│   ├── page.tsx      # Home page
│   ├── globals.css   # Global styles
│   ├── robots.ts     # Robots.txt generator
│   └── sitemap.ts    # Sitemap generator
├── components/       # React components (to be created)
├── lib/              # Utility functions (to be created)
└── public/           # Static assets
```
